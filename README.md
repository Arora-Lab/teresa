# Teresa Family Charity

A modern bilingual platform for Teresa Family Charity.

## Mission

Provide an easy way for donors, volunteers, and administrators to support elderly individuals in Vietnam while maintaining transparency and long-term sustainability.

## Technology

- Next.js
- ASP.NET Core 9
- PostgreSQL
- Sanity CMS
- Tailwind CSS
- Stripe
- PayPal
- Cloudflare
- Docker

## Features

- 🌎 English & Vietnamese
- 💳 Online donations
- 📰 Blog & News
- 📷 Photo galleries
- 🎁 Donor portal
- 🧾 Tax receipts
- 📊 Admin dashboard

## Roadmap

- Phase 1 — Static website
- Phase 2 — CMS
- Phase 3 — Donations
- Phase 4 — Admin
- Phase 5 — AI Assistant
- Phase 6 — Mobile App
