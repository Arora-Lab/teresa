# Project Roadmap

This document outlines the high-level roadmap for the Teresa Family Charity platform.

## Milestones

### v0.1.0: Static Website
- Setup bilingual Next.js skeleton.
- Implement premium design system.
- Build static pages (Home, Mission, Formation, Contribute, Contact).

### v0.2.0: CMS Integration
- Integrate Sanity CMS.
- Create dynamic Blog, News, and Gallery sections.

### v0.3.0: Donations Processing
- Integrate Stripe and PayPal.
- Implement secure, PCI-compliant donation flows.

### v0.4.0: Donor Portal
- Integrate Clerk Authentication.
- Allow users to view donation history and generate tax receipts.

### v0.5.0: Admin Dashboard
- Custom management portal for charity administrators.
- Volunteer coordination tools.

### v1.0.0: Production Launch
- Final security audits.
- Production deployment to live domain.
